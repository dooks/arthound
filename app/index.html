Help

Uses simple boolean to search [AND, OR, NOT]. Booleans consists of [term] [boolean] [term]. Booleans are case insensitive.

For quality searches, the site automatically pulls from popular submissions for each source first (score quota, minimum number of views, etc). This can be disabled in options.

If a tag cannot be found, Deviant Art is persistent in returning something.

e926 does not allow more than 5 total search terms without authentication and will not return anything if this occurs. Search terms do not include booleans.

Limit controls how much is returned per source. Imgur does not provide limit options, and will return more than this limit.

Due to its API credit use, response time for returning searches from Imgur will diminish until reset after 24 hours.

Search Examples

  • cats AND dogs NOT anthro
  • steven_universe AND rose or peridot
  • charizard or blastoise
  • sora not donald not goofy
